# 預選休假｜順位前統計工具

純前端工具（單一 `index.html`）。上傳每月預選休假的 Excel（Google 表單回覆匯出檔），選擇自己的名字，即可看出排在你前面的人把哪些日子選走了。

- 🔒 檔案只在瀏覽器內處理，**不會上傳到任何伺服器**。
- 可直接雙擊 `index.html` 開啟。

## 日期排除規則（關鍵字／刪除線）

解析儲存格日期時，會依下列規則自動排除：

| 觸發條件 | 判斷層級 | 作廢範圍 |
|---|---|---|
| 含 `暫停` | 逐格 | **只作廢含此字的那一格**（其餘假別欄仍有效） |
| 含 `逾時` | 整列 | **所有假別欄（預選休假／特休／福利補休／加班補休…）全部作廢** |
| 含 `已超過` | 整列 | 全部作廢 |
| 含 `不計算` | 整列 | 全部作廢 |
| 含 `不受理` | 整列 | 全部作廢 |
| 姓名格被劃刪除線 | 整列 | 整筆作廢（不佔順位、不進名單） |
| 整格被劃刪除線 | 逐格 | 該格所有日期排除 |
| 格內**部分**文字被劃刪除線 | 逐格 | 只排除被劃掉的那幾天，保留其餘 |

> 整列作廢的關鍵字定義於 `isOverdueRow()`；逐格的 `暫停` 定義於 `cellDates()`。
> 設計差異：`暫停預選休假` 常見於「F 欄（預選休假）暫停、但 G/H 欄仍有有效的改本特休」，故只作廢單格；
> `逾時／已超過／不計算／不受理` 代表整筆逾時無效，故全欄作廢。

## 其他解析行為

- **日期格式**：辨識 `M/D`（如 `7/4`、`7/4 (星期六)`）。點／減號分隔的簡寫（如 `14.22.23`、`6-8`）不會被當成日期。
- **去重**：以 `email > 工號 > 姓名` 為鍵，保留**時間最新**的一筆；順位 = 原始填表列序。
- **重複日期**：同一格出現同一天多次只計一次。
